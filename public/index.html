Halo, Saya kanawangyy!

Developer di balik AniZone 2026 — platform streaming anime sub indo yang dibangun dengan semangat. Proyek ini adalah hasil kerja keras belajar web development.

Tech Stack

Node.js + Express (backend), Firebase Auth & Firestore (database), MyAnimeList API (data anime), dan Vercel (hosting).

Komunitas

Bergabung dengan komunitas AniZone di WhatsApp Channel untuk update terbaru dan rekomendasi anime.

GitHub WA Channel

GET /api/latest

Anime terbaru. Query: ?page=1. Return: title, url, image, episode.

GET /api/search

Pencarian anime. Query: ?q=naruto. Return: hasil dengan score dan tipe.

GET /api/detail

Detail anime + episode. Query: ?url=.... Include synopsis dari MAL.

GET /api/watch

Link streaming episode. Query: ?url=.... Return: daftar server.

GET /api/trending

Anime trending dari MyAnimeList. Return: ranking, score, genre.

GET /api/schedule

Jadwal rilis musim ini dari MAL, dikelompok per hari tayang.

GET /api/news

Berita anime terbaru. Return: title, deskripsi, tanggal, link sumber.
