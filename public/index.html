Tentang AniZone

Web App Streaming Anime resolusi HD tanpa iklan pop-up mengganggu. Didesain dengan standar PWA (Progressive Web App) eksklusif tahun 2026 yang dapat diinstal langsung di perangkat mobile.

Fitur Unggulan

Menggunakan IndexedDB untuk penyimpanan riwayat tontonan dan favorit secara lokal dengan aman. dan real-time fetching data.

🛠 Tech Stack

Vanilla JavaScript (ES6+), HTML5, CSS3 dengan fitur modern seperti CSS Grid, Flexbox, Custom Properties, dan animasi keyframe.

⚡ PWA & Performance

Service Worker untuk caching offline, IndexedDB untuk storage lokal, dan lazy loading gambar untuk performa optimal di semua perangkat.

🎨 Design System

Material You inspired dengan dark/light theme toggle, CSS variables, responsive layout dengan sidebar desktop dan bottom nav mobile.
